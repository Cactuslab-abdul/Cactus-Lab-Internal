# Operations

## Monthly Client Workflow

| Week               | Task                                                         |
| ------------------ | ------------------------------------------------------------ |
| Week 1             | Content planning call (20 min max) → Script all 15 videos    |
| Week 1–2           | Batch all filming into 1–2 shoot days per client             |
| Rolling            | Send raw footage to editor within 24hrs of shoot day         |
| Rolling            | Editor turnaround: 48hrs per video                           |
| Rolling            | Review and approve all edits within 24hrs of receiving       |
| End of month       | Schedule all 15 posts via Later/Buffer — never post manually |
| Day 3 (next month) | Send monthly performance report to client                    |

**Daily:** Story reposts — max 15 mins per client per day
**Bi-weekly:** Short performance update (views, followers, top post)
**Monthly:** 20-min check-in call (results + upcoming plan + feedback)

## Quality Control

**Pre-publish QC checklist — every video, no exceptions. Do not publish if any item fails.**

| Check | Pass Criteria | Fail = Do This |
|-------|--------------|----------------|
| Hook timing | A visual cut, text, or spoken hook appears within the first 2 seconds — no black screen, no logo intro, no music fade-in | Return to editor with timestamp note |
| Captions accuracy | Play through at 1x speed and read every caption word — no misspellings, no wrong words, no missing lines | Return to editor with corrected caption file |
| CTA present | Last 3 seconds contain at least one call to action (follow, comment, DM, visit link in bio) — must be visible on screen or spoken | Add CTA text overlay before publishing |
| Aspect ratio | Video is 9:16 (1080×1920px) for Reels/TikTok/Shorts; no black bars on sides or top/bottom | Return to editor — wrong export settings |
| Audio levels | No clipping, no sudden loud spikes, music doesn't drown out voiceover | Return to editor for audio mix fix |
| Branding | Client's handle/logo watermark present if required by client brief | Add in Canva or CapCut before publishing |
| Platform suitability | No competitor platform logos visible (e.g. TikTok watermark on a Reel) | Remove watermark — re-export clean |

**Revision policy:** 1 round per video — written into every contract. Never give unlimited revisions.

### For Every Video — Minimum Required Shots

| Shot | Description | Filmed? |
|------|-------------|---------|
| Hook shot | The very first frame — grabbing attention. A close-up, a movement, a reaction, or text reveal. Must be captivating within 0.5 seconds. | [ ] |
| Wide/establishing shot | Shows the space, product, or person in context | [ ] |
| Close-up detail shot | Product close-up, hands, face reaction, or key detail | [ ] |
| Action shot | Something happening — pouring, opening, driving, demonstrating | [ ] |
| Talking head (if applicable) | Presenter to camera, tight framing, clean background or branded space | [ ] |
| CTA shot | Final frame — text overlay, product logo, or spoken call to action | [ ] |

---

### Video-Type Shot Guides

**Product showcase (perfume, watches, food, pet products)**
- [ ] Hero shot: product in frame alone, clean background, good light
- [ ] Hands-on: person picking up, opening, using the product
- [ ] Detail close-up: label, texture, packaging detail
- [ ] Lifestyle shot: product in real-life setting (shelf, table, hands)
- [ ] Price/value reveal shot (if used in script)
- [ ] Reaction shot: person's face seeing/using the product

**Location/store walkthrough (car garage, shop, restaurant)**
- [ ] Exterior establishing shot — wide, shows the building or signage
- [ ] Interior walkthrough — slow pan or walking shot
- [ ] Hero asset shot — the best car, dish, or product in the space
- [ ] Staff/owner doing their thing — candid or directed
- [ ] Close-up of one product or detail that tells the story
- [ ] Exit/outro shot — satisfying close

**Talking head / educational video**
- [ ] Framing check: subject fills frame from chest up, background is clean/branded
- [ ] Take 1: straight to camera, no warm-up
- [ ] Take 2: same script, different energy
- [ ] B-roll for cutaways: relevant product shots, screen recordings, or environment
- [ ] Cutaway reaction or demonstration if the script references anything

**Before/after (grooming, detailing, cleaning, renovation)**
- [ ] Before: same angle, same light if possible — get wide AND close
- [ ] Process: one or two action shots showing the work
- [ ] After reveal: same angle as before — this is the money shot, get multiple takes
- [ ] Reaction shot: person seeing the result

---

### On-Set Reminders

- Check audio before every take — tap the mic, listen back on earphones
- Keep background clean — remove clutter, cables, irrelevant branding
- Shoot in 4K or highest quality available
- Shoot at 60fps for anything with movement — allows slow-mo in edit
- Get 2–3 takes of every key shot — editing options save time in post
- Name files immediately after shoot: `ClientName_VideoTitle_ShootDate_v1`

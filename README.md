# Cactus Lab — Shared Vault

Shared Obsidian vault for Cactus Lab.

## Workflow

1. Before starting a session: `git pull`
2. Work as normal — Claude writes updates here on `/wrapup`
3. After `/wrapup`: `git push`

## Structure

- `Dashboard.md` — current status overview
- `Sessions/` — per-session logs (auto-generated by Claude on wrapup)
- `Strategy/` — roadmap, positioning, decisions
- `Projects/` — one folder per active project
- `Meetings/` — meeting notes
- `Resources/` — research, references, assets

# Sales System

## Pricing

- **Anchor price: AED 5,500/month** — never open below this
- Discount is your closing lever, not your opening price (minimum: AED 4,500)
- Monthly revenue target = AED 5,500 × number of clients

---

## Lead Generation

**Daily targets:** Find 40–50 leads, message 30 minimum

**Instagram setup (must be done before any outreach):**
- 12+ posts live before first DM
- Bio: *"We make short videos for UAE businesses. You stay off camera."*
- Highlight reel: Pets Delight results + BTS clips

**Prospecting channels:**
- Instagram hashtags per niche: `#dubaigarage` `#dubaiperfume` `#dubairecruiter` `#dubaiwatch` `#dubaispices`
- Google Maps saved searches per niche in Dubai and Sharjah
- Walk-in area rotation: Al Quoz (Mon), Deira (Tue), Sharjah Industrial (Thu)

**WhatsApp Business:**
- Profile: Cactus Lab name, logo, business description
- Business hours set (9am–7pm, Mon–Sat)
- Away message for outside hours
- Label leads by stage: New / Replied / Proposal / Client

---

## Outreach Scripts

**Block 9am–10:30am daily — outreach only, no client work in this window.**

Write 3 DM openers, one per niche cluster:
1. Perfume/watch niche
2. Car niche
3. Recruitment niche

Structure: personalised first line (reference something specific) + clear hook

**WhatsApp cold opener template** for walk-in follow-ups — keep it in saved messages.

**Rule:** Never DM from a hollow account — verify your profile looks credible before each outreach session.

## Follow-Up Cadence

| Day | Action |
|-----|--------|
| Day 0 | Send initial DM/message |
| Day 3 | *"Just checking if you had a chance to see this"* |
| Day 7 | *"Last follow-up — happy to show you what we did for a similar brand"* |
| 2 no-replies | Move to "30-day warm-up" list — re-engage next month |
| Every Friday | Review all "Replied" leads and push every open conversation forward |

**Log every lead in the pipeline tracker immediately after messaging** (`Operations/cactus_lab_pipeline_tracker.html`)

## Closing Playbook

**WhatsApp pitch structure:**
> Problem → Solution → Proof → Offer → 48-hour close

**Proof folder (Google Drive):**
- Pets Delight content samples
- Metrics screenshots
- Account growth stats

**3 objection responses to prepare:**
1. *"Too expensive"* → anchor back to ROI, offer phased payment
2. *"Let me think about it"* → set a 48hr decision deadline
3. *"We already do it in-house"* → compare output quality/volume

**Closing rules:**
- Close via WhatsApp voice note or call — never a text thread
- Set a 48-hour decision deadline on every verbal agreement
- Anchor at AED 5,500 — discount only to close, not to start

## Referral System

- Ask Pets Delight for 3 warm introductions this month
- Referral incentive: 1 free month for each referred client who signs
- After every new closed deal: ask within 2 weeks if they know anyone who'd benefit
